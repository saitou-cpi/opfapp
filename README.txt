## Setting
sudo dnf install -y python3-pip git
pip install virtualenv

git clone https://github.com/saitou-cpi/opfapp.git
cd opfapp/
virtualenv .venv
source .venv/bin/activate
pip install -r requirements.txt
sudo chmod a+x run.sh

gunicorn -w 4 -b 0.0.0.0:8000 app:app

## 今後の改善
更なるバックテスト期間の追加:
1週間だけでなく、1ヶ月や3ヶ月など複数の期間での検証も行えるようにすることで、パラメータの確実性をさらに高めることができます。

複数の銘柄の同時最適化:
複数の銘柄の最適化を同時に行う機能を追加することで、ポートフォリオ全体の最適化が可能になります。

可視化機能:
売買履歴や価格の推移をグラフで表示することで、直感的に結果を理解しやすくなります。

リスク管理:
各取引のリスク評価を行い、リスクが高い取引を避けるためのフィルタリング機能を追加します。
